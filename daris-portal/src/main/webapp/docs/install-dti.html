The application uses Arcitecta Desktop Integration, which is a Java applet, to access the local file system as well as communicate with the remote Mediaflux server. It is requires by various functions, e.g. data importing, local file browsing.

  • To make DTI applet work:
    1. Make sure you have Java installed.
    2. If you have already installed Java VM (JRE), but the Desktop Integration applet is still not loaded in your browser, you can:
      • Go to http://www.java.com/en/download/testjava.jsp
      • If the LIVE area at the top of the page is showing Inactive Plug-in, click that area, it will prompt a popup window ask for trust confirmation, read and trust it. You can then load Java applets in Safari.
    3. The DTI writes to log files in ~.Arcitecta/DTI/logs/ directory. You can look at the logs to see if there is anything preventing the DTI from being started.
